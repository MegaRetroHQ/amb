## 📋 Чеклист компонентов

### Agents API

- [ ] `POST /api/agents` — создание
- [ ] `GET /api/agents` — список
- [ ] `GET /api/agents/search?q=` — поиск
- [ ] Дубликаты имен (документировать поведение)

### Threads API

- [ ] `POST /api/threads` — создание
- [ ] `GET /api/threads` — список
- [ ] `PATCH /api/threads/:id` — закрытие
- [ ] `DELETE /api/threads/:id` — удаление
- [ ] Закрытый тред не принимает сообщения

### Messages API

- [ ] `POST /api/messages/send` — отправка
- [ ] `GET /api/messages/inbox?agentId=` — входящие
- [ ] `POST /api/messages/:id/ack` — подтверждение
- [ ] Broadcast (`toAgentId: null`)
- [ ] Replies (`parentId`)

### DLQ API

- [ ] `GET /api/dlq` — список
- [ ] `POST /api/dlq/:id/retry` — повтор одного
- [ ] `POST /api/dlq/retry-all` — повтор всех

## 🔍 Граничные случаи

| Случай | Ожидаемый результат | Статус |
|--------|---------------------|--------|
| ACK `pending` сообщения | 409 `conflict` | ⬜ |
| ACK несуществующего сообщения | 404 `not_found` | ⬜ |
| Двойной ACK | Идемпотентен (200) | ⬜ |
| Отсутствует `name` при создании агента | 400 `invalid_params` | ⬜ |
| Несуществующий `threadId` | 404 `not_found` | ⬜ |
| Отправка в закрытый тред | 409 или валидация | ⬜ |
