## 🎯 Выполненные задачи

### 1. Проверка inbox функциональности

**Результат:** Обнаружена и исправлена критическая проблема (P0)

**Найденные проблемы:**
- ❌ Inbox не обрабатывал broadcast сообщения (`toAgentId = null`)
- ❌ Inbox не исключал собственные сообщения агента

**Исправления:**
- ✅ Добавлена поддержка broadcast сообщений через `OR: [{ toAgentId: agentId }, { toAgentId: null }]`
- ✅ Добавлено исключение собственных сообщений через `fromAgentId: { not: agentId }`

**Измененные файлы:**
- `lib/services/messages.ts` — функция `getInboxMessages()`
